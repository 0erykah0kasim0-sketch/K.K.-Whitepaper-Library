# 01_OS-Architecture — 作業系統架構層（Operational System Architecture）

本資料夾收錄 K.K. Universe 中所有與「多域作業系統（OS）」相關的核心概念。
此層屬於文明工程（Civilization Engineering）的系統框架級別，涵蓋：

- ND-OS（Natural Denial Operating System）
- Island-OS（Island-State Multi-Module OS）
- Meta-OS（Higher-order OS，文明級統合框架）
- 子模組（Modules）、子系統（Sub-systems）、場域耦合（Field Integration）

此層描述「文明如何透過作業系統思維來理解世界、運作世界、維持韌性」。
