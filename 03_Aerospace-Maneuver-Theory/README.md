# 03_Aerospace-Maneuver-Theory — 空域操作宇宙層（Aerospace Maneuver & Air-Domain Theory）

本資料夾收錄 K.K. Universe 中所有與「空域作戰、動態幾何、反追蹤、向量機動」相關的高階理論。
包含 FlashNet、Angle-Flash、Reverse-ECS、TSAC 等跨域空域模型。

此層定義未來空域的運作方式，並提出超越傳統制空概念的全新空域哲學。

## 核心主題（Examples）
- FlashNet Air-Denial System（閃現式空域拒止系統）
- Angle-Flash Maneuver（角度閃現攻防邏輯）
- Reverse-ECS（反向電子遮罩，Coak & Scatter 模型）
- TSAC — Temporal-Spatial Air Combat（時間—空間空戰框架）
- Terrain Vector Geometry（地形向量幾何）
- Dead-Zone Lurking（EMCON Zero 概念）
- Anti-BVR Maneuver Modeling（反超視距追蹤模型）
- Flash-Chain Strike Cycle（閃鏈攻擊循環）
- 空域中的混沌、相態、梯度耦合
