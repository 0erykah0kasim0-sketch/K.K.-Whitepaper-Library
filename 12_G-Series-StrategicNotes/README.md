# 12_G-Series-StrategicNotes
G-Brief｜G-Analysis｜G-Insight 系列（戰略短評 / 概念解析）

此宇宙層收錄 K.K. Universe 中所有「高頻短篇戰略思想」：
包括空域文明、無人化文明、能源文明、後勤文明、場域升空哲學等。

特性：
- 高密度、短篇、模型導向
- 安全、抽象、概念層
- 不屬於白皮，也不是 DARPA Notes
- 類似 RAND / CSBA 的 Strategic Insight 格式

子分類如下：
- AIR/   （空域文明）
- UAV/   （無人化文明）
- AERO/  （升空與場域文明）
- ENE/   （能源文明）
- LOG/   （後勤文明）
- CIV/   （敘事、文明層解析）

每個 G-Series 檔案皆使用：
<CODE>–G1–<DOMAIN><編號>–v1.0.md

此層作為「戰略哲學 × 系統行為 × 概念模型」的專屬宇宙。
