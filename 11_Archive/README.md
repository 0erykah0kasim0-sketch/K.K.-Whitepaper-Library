# 11_Archive — 暫存 × 草稿 × 歷史版本層
(Archive for Drafts, Legacy Files & Unsorted Materials)

本資料夾用於收納尚未分類、暫時放置、未定義、或舊版的白皮內容。
此層作為 K.K. Universe 的「安全緩衝區（Buffer Layer）」，避免主宇宙層混亂。

## 分類原因
K.K. 的白皮世界龐大且跨域，Archive 用於保持整體結構乾淨：
- 尚未確定分類的白皮
- 草稿（draft）
- 早期版本（legacy）
- 名稱未定義的想法
- 初次推論、碎片化推演
- 未完全成形的世界線或物理模型

## 推薦用途
- 作為「待整理」區域（To-be-sorted）
- 保存草稿，未來可升級至正式宇宙層
- 臨時匯入新白皮，再決定放哪裡
- 保存已被新版取代的舊版本

## 命名建議
暫時可以先用：
- <name>_draft.md
- <concept>-temp.md
- <year>-legacy.md

等未來分類明確後再移動至主宇宙層。

## 注意
Archive 的存在是為了：
- 保持整個 Library v2.0 的乾淨與可擴張性  
- 讓哥哥可以自由創作，不需一開始就分類  
- 讓未來世界線擴張時有緩衝與調整空間
