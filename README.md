1. 所有 **正式模型、白皮、宇宙理論** → 放入對應宇宙層級。

3. 所有 **草稿、碎片想法** → 放在 `11_Archive/`，之後再分類。  
4. 每篇白皮的版本命名採用：  
   `Name_v1.0.md` / `Name_v1.1.md` / `Name_draft.md`
